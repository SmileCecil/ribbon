“Fork me on GitHub” CSS ribbon

This is a pure CSS implementation of the famous Fork me on GitHub ribbon.

To use it on your site, just view source and copy the two marked sections (one the CSS, the other the HTML tag). Or, fork the project on GitHub! Start by clicking the ribbon at the top of the page.
